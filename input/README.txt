Директория input: Правила для файлов, подлежащих обработке

В директории "input" должны находиться только те файлы, которые будут подвергнуты следующим изменениям в папке "output":

1. Код будет преобразован в соответствии с шаблоном.
2. Имя файла будет изменено в соответствии с названием файла.

Чтобы файл попал в директорию "input" и был обработан, он должен соответствовать следующим критериям:

1. Внесенные изменения будут основаны на шаблоне кода.
2. Имя файла в "output" будет изменено в соответствии с названием файла из директории "input".
